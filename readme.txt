Allows you to easily require SSL for specific pages.

== Description ==
This plugin allows you to force certain pages to display as SSL-only by simply checking a box. Please note, you must have SSL configured on your server and require SSL for either login or admin.


== Installation ==

1. Upload the plugin to your plugins directory
2. Activate the plugin
3. Make sure either FORCE_SSL_LOGIN or FORCE_SSL_ADMIN (or both) is defined true in your wp-config.
4. Check the box on the edit page screen to force it to be ssl.
